รูปตนเอง

เหตุผลที่อยากเรียน Full stack web development

ต้องการเรียน Full Stack Web Development เพราะต้องการมีความสามารถในการพัฒนาเว็บทั้งฝั่ง Front-end และ Back-end โดยที่ไม่ต้องพึ่งพาผู้อื่น การมีความเชี่ยวชาญทั้งสองด้านจะช่วยให้ผมสามารถสร้างเว็บแอปพลิเคชันที่สมบูรณ์และสามารถทำงานได้อย่างมีประสิทธิภาพ นอกจากนี้ ความรู้ในด้านนี้ยังเปิดโอกาสในการทำงานในวงการ IT อย่างกว้างขวาง และยังช่วยพัฒนาทักษะการแก้ไขปัญหาอย่างเป็นระบบอีกด้วย

ความคาดหวังต่อวิชานี้

ผมคาดหวังว่าจะได้รับความรู้และทักษะที่จำเป็นในการพัฒนาเว็บแบบ Full Stack เรียนรู้การใช้เครื่องมือและเทคโนโลยีที่สำคัญในวงการ เช่น HTML, CSS, JavaScript ผมหวังว่าวิชานี้จะช่วยให้ผมสามารถสร้างโปรเจกต์จริง ๆ ได้ และเพิ่มความมั่นใจในการนำทักษะเหล่านี้ไปใช้ในงานจริงหรือโปรเจกต์ส่วนตัว

จุดอ่อนที่ต้องพัฒนาเกี่ยวกับทักษะ Programming และ แนวทางการพัฒนา

จุดอ่อนของผมในเรื่อง Programming คือ การจัดการกับโค้ดที่ซับซ้อนและการ Debug โค้ด เมื่อมีปัญหาในโค้ดที่มีความซับซ้อน ผมมักจะใช้เวลานานในการหาสาเหตุและวิธีแก้ไข อีกทั้งบางครั้งผมยังขาดความรู้ในเชิงลึกเกี่ยวกับการออกแบบสถาปัตยกรรมซอฟต์แวร์ แนวทางการพัฒนาคือ ผมจะพยายามฝึกฝนการเขียนโค้ดให้มากขึ้น โดยเฉพาะอย่างยิ่งการแก้ปัญหาผ่านการทำโปรเจกต์จริง ๆ นอกจากนี้ ผมยังต้องการเรียนรู้และทำความเข้าใจเกี่ยวกับแนวคิดเชิงสถาปัตยกรรมของซอฟต์แวร์ให้มากขึ้น ผ่านการศึกษาเพิ่มเติมจากแหล่งข้อมูลต่าง ๆ และการเข้าร่วมชุมชนนักพัฒนาเพื่อแลกเปลี่ยนความรู้และประสบการณ์
